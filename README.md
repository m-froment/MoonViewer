### Moon Viewer app 

An small app to visualize the visibility of the Moon at different locations on the Earth, and its aspect from several observatories. The app is written using the <code>Streamlit</code> Python library.  

To run it, write: 
```
streamlit run app.py
```
Or simply go to: 
<br><br>
https://moon-viewer.streamlit.app/ <br><br>
